Crystal Collector

How to play this game:

  • A random number will be displayed at the start of the game
  • Clicking on one of the crystals below will add a specific number of points to the total score
  • The value of the crystal is hidden
  • The value of the crystal will change each time the game starts
  • The game is won by the total score matching the random number
  • The game is lost if the total score exceeds the random number
